# pink-voice

Voice input daemon with configurable global hotkey. Requires pink-transcriber running.

```bash
{{PINK_TOOLS}}/pink-voice/pink-voice            # Start daemon
{{PINK_TOOLS}}/pink-voice/pink-voice stop        # Stop daemon
{{PINK_TOOLS}}/pink-voice/pink-voice status      # Check if running
{{PINK_TOOLS}}/pink-voice/pink-voice settings    # Configure hotkey, sounds, prefix
```

Hotkey (default: Alt+Q) starts/stops recording. Result copied to clipboard.
